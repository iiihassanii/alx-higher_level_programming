The my_list module
======================
Using ``print sorted list``
-------------------
import the model
	>>> MyList = __import__('1-my_list').MyList


Now use it:
Adding five elements to MyList instance.


can sort and print an empty MyList instance

send bool and int
    >>> test = [True, 2, 5]
    >>> my_list = MyList(test)
    >>> my_list.print_sorted()
    [True, 2, 5]

negative numbers
    >>> test = [-1, -2, -3]
    >>> my_list = MyList(test)
    >>> my_list.print_sorted()
    [-3, -2, -1]

negative numbers
    >>> test = []
    >>> my_list = MyList(test)
    >>> my_list.print_sorted()
    []
